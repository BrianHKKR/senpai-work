# Senpai.work — プロジェクト概要

## コンセプト
日本在住のシニア人材（50-70代）に、外国法人/個人のビジネス実務を案件単位で依頼するマーケットプレイス。

## 背景 (3호기 연결)
- 化粧品OEM工場 인수 (BATONZ #45497 등) → 현장 방문·오너 미팅 대행 필요
- 나고야 부동산 (와비사비 라운지) → 긴급연락처·내견 대행 필요
- Mement Japan GK → 거주 디렉터 후보 풀 필요
- 1차 클라이언트 = Brian 본인

## 業務ティア
- T1 (¥5,000-8,000): 書類サポート — 緊急連絡先、郵便物、役所書類
- T2 (¥8,000-25,000): 現場代行 — 工場/物件視察、内見、写真レポート
- T3 (¥25,000-80,000): 交渉・面談 — M&Aオーナー面談、不動産交渉
- T4 (月額制): ディレクター — 法人役員、契約署名

## 決済
- Wise (HK法人 → 日本個人銀行口座)
- 業務委託契約 (雇用関係ではない)
- 年間¥200,000以下 → 確定申告不要の範囲で運用
- 1件¥1,000,000未満 → 国外送金等調書の対象外

## スクリーニング
- Step 1: オンラインフォーム (基本情報)
- Step 2: KYC (運転免許/マイナンバーカード)
- Step 3: ビデオ面談 (LINE/Zoom, 10-15分)
- 信頼等級: 新規 → 認証 → シニア → プレミアム (実績ベース昇格)

## 技術スタック
- Next.js + TypeScript + Tailwind CSS
- ホスティング: Netlify (予定)
- ドメイン: senpai.work

## 競合分析
- シルバー人材センター: 単純作業のみ、ビジネス代行なし
- CrowdWorks/Coconala: シニア特化なし、現場代行なし
- ノミニーディレクターサービス: 高コスト、固定契約
- → 「信頼できるシニアが建当たりで現場実行」= 空白ポジション

## 法的要件
- 業務委託契約書テンプレート (日本語)
- プライバシーポリシー (個人情報保護法)
- 特定商取引法に基づく表記
- 運営法人: Mement Hong Kong Limited
